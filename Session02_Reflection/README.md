The following examples show how to use basic and more advanced features of the Reflection in C#.

#### Example01

This example describes how to work with `System.Type` instance which encapsulates the information about a certain type (metatype) as well as structures used to represent information about constructors, methods and method paramteres.  

#### Example02

This examples shows how to work with the reflection to create instances of the given type in the runtime as well as dynamically set properties' values.

#### Example03

This examples shows how to define custom attributes and how to use them with reflection.
(Simple - assign different name to the property/class, complex - add custom behavior to method or even add Func type to the property)


#### Example04 (extra)
This examples shows how to parse expression using Visitor pattern which can be useful when fetching data from database in ORM implementation.
